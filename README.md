# Retail Sales Analytics Dashboard

## Tools Used
- Python (Pandas, NumPy)
- MySQL
- Power BI

## Project Overview
This project analyzes retail sales data using Python for data cleaning and transformation, stores the processed data in MySQL, and visualizes insights using Power BI.

## Key Features
- KPI cards for Sales, Profit, and Profit Margin
- Monthly sales trend analysis
- Category and region-wise performance
- Interactive slicers for filtering
